# USMLE Step 1 — Hematology & Oncology Guide

This repository is ready for GitHub Pages.

## Files

- `index.html` — the complete single-page website.
- `.nojekyll` — tells GitHub Pages to serve the static site directly.

## How to publish on GitHub Pages

1. Create a new GitHub repository.
2. Upload the contents of this folder to the root of the repository.
3. Go to **Settings** → **Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Choose branch **main** and folder **/root**.
6. Save.
7. GitHub will provide your live website URL after deployment.

## Notes

The site is a static HTML file. It does not require a backend, database, or build process.
